# Table
Tables are styled by adding the class `table`.

### Example: Basic

{% example html %}
<table class="table">
  <thead>
    <tr>
      <th>Column 1</th>
      <th>Column 2</th>
      <th>Column 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>One line item</td>
      <td>One line item</td>
      <td>One line item</td>
    </tr>
    <tr>
      <td>One line item</td>
      <td>One line item</td>
      <td>One line item</td>
    </tr>
  </tbody>
</table>
{% endexample %}
